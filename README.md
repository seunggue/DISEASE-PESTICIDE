# 자율프로젝트

### 프로젝트 소개

- Flutter를 이용한 병해충 정보 시스템
- 공공데이터를 이용한 병해측 예측 지도
- 대표 64작물에 관한 모든 병해충 저보를 제공하고 해당 병해충에 대한 생태환경, 대비방안, 증상 등의 정보 제공
- 대표 64작물의 병해충과 관련된 농약 정보를 제공
